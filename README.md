# Task Manager API (Node.js)

Un'applicazione backend costruita con Node.js per la gestione di attività e utenti. Questo progetto fornisce un'API RESTful completa per creare, leggere, aggiornare e eliminare (CRUD) task e le note collegate a un task, con supporto per l'autenticazione degli utenti.

## 🚀 Funzionalità

- **Autenticazione Utente**: Login con JWT (JSON Web Tokens).
- **Gestione Task**: Ogni utente può gestire i propri task in modo isolato.
- **Operazioni CRUD**: Gestione completa di profili utente e attività.
- **Sicurezza**: Password criptate nel database (Bcrypt).

## 🛠️ Tecnologie utilizzate

- **Node.js**: Ambiente di runtime JavaScript.
- **Express.js**: Framework web per le API.
- **MongoDB**: Database NoSQL (con Mongoose ODM).
- **JWT**: Per l'autenticazione sicura.
- **Bcrypt**: Per l'hashing delle password.
